**Founder doctrine, verbatim (2026-07-13):** "In season / Out of season / Every season / No season /
Strategies. Learn to identify which is which and where and when they fit and you will be a profitable
trader for life."

This is the ORGANIZING FRAMEWORK for the whole trading operation — it replaces the dead "find the one
true strategy" model (the fade doctrine that collapsed 2026-07-13, see [[project-mym-fade-doctrine-verdict]]).
**The durable edge is not any strategy — it's the classification skill.**

**The four seasons:**
- **NO SEASON** = never worked for anyone, any regime — FICTION (fill look-ahead / un-fillable). e.g. the
  entire QF-fade catalog: the +$246k was prices the tape never offered. Bury forever, never revive.
- **EVERY SEASON** = survives all history/regimes. Rare, and therefore MODEST ("all-weather" and "spectacular"
  almost never coexist). e.g. the ORB/momentum book (PF~1.2). The boring always-there base.
- **IN SEASON** = strong in the CURRENT regime, honestly fillable, but not eternal. Trade it while it works.
- **OUT OF SEASON** = had a season, lost it, dormant — bench with a RETURN TRIGGER, watch for its regime.

**Why: honest fills is the GATE to the whole taxonomy.** It is the only thing separating NO-season fiction
from the three real seasons — you cannot read a strategy's season until you've stripped the fills-that-never-
existed, or every mirage reads "in season." (The all-history robustness bar was the WRONG filter — it rejects
tradeable IN-season edges for not being EVERY-season; only honest-fills is non-negotiable.)

**The hard part = the classification itself.** Telling a real IN-season edge from RECENT NOISE is a
multiple-testing knife-edge (K≈13k tested → ~650 fake "in season" by chance). So the doctrine is right and
the DISCIPLINE earns the classification: honest fills + multiple-testing correction + forward decay-monitors.

**How to apply — build the SEASON ENGINE (not another strategy):** tag every strategy in/out/every/no;
trade IN+EVERY (sized to survive, see [[project-mym-fade-doctrine-verdict]] roadmap — size to survive not
pass fast); bench OUT with a return trigger; bury NO; live decay-monitors move strategies between seasons
automatically (flag when an IN-season edge cools, when a dormant one's regime returns). First live instance:
the current-edge-finder + graveyard-resurrection searches (2026-07-13). Related: [[feedback-backtest-measures-robot-not-trader]],
[[project-mym-strategy-search-verdict]].
