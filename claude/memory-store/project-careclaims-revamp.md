Full revamp of careclaimsadjusting.com (the user's already-premium 139-page site) using the [[project-image2code-pipeline]] strategy. Goal: level it up DRAMATICALLY, not parity. Build dir: `/tmp/cca/site/`; real repo: `~/code/projects/careclaimsadjusting-website/`.

**Architecture:** one shared design system (compiled `tw.css` ~26KB via play-CDN render-extract, `cca.css` premium/motion, `cca.js` canvas-particle/magnetic-cursor/kinetic/reveals) + 6 localized chunks (header/footer/trustbar/leadform/progress/aurora) → applied site-wide. Every page ports its original SEO `<head>` VERBATIM (ld+json/canonical/hreflang/og/Speakable) — exact parity verified vs live.

**Three deterministic engines** (`/tmp/cca/site/`), all locale-aware via `CCA_LOCALE=en|es`:
- `fill_city.py` + `templates/city.body[.es].html` — bespoke tokenized template for the 20 uniform city pages (the standout; beats live's off-brand stock-photo cities).
- `fill_structured.py` — splits original body on `<h2>`, detects shape per section (≥2 `<h3>`→card grid w/ emoji icons; `<ol>`→stepper; `<ul>`→feature checklist; FAQ→accordion), alternating bands. Use for CARD-GRID pages (services, 7 damage pages, contractors, claimpilot, how-we-work). **The prose engine flattened these into ugly text dumps — that was the one place the rebuild REGRESSED vs live; structured engine fixed it.**
- `fill_content.py` — content-preserving re-skin (lift body verbatim, strip old chrome/classes/inline-style, premium typography). Use for PROSE pages (18 articles w/ E-E-A-T byline + featured-image card, 9 case studies, one-offs).

**Hard-won gotchas:** (1) homepage's 3 Image-2.0 assets were relative-path refs → 404'd in deploy ("images disappeared"); all asset refs must be absolute `/assets/`. (2) ES string-translation order: multi-word phrases ("Schedule Your Free Inspection") MUST replace before substrings ("Free Inspection"). (3) count-up stats rest at 0 in static screenshots — animate live, not a bug. (4) Tailwind must be compiled site-wide (union of ALL page bodies) or inner-page classes go missing.
