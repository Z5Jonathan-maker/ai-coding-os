For digital-product PDFs (Gumroad line, dosecraft-companion):

**Be bold with the conversion-driving framing.** Use "reverse" / "remission" /
"resolve" / "restore" — hedged once with Path B verbs ("research suggests,"
"the literature describes," "trial data describes," "animal-model research
reports"). That single hedge IS the editorial safety layer. The user has
attorney-confirmed this framing is sufficient.

**Don't stack hedges.** Avoid:
- Hedging within hedges ("research-suggested in a fraction of cohorts")
- Hedge-then-deflate ("can drive remission... in some subset")
- Over-deferring to standard-of-care ("not a replacement for prescribed therapy")
  on every single section
- Multiple "critical note" sections per pack
- Cautionary framing that buries the conversion claim

**Why:** User explicitly said "obv the cures, reverses, etc language is the
biggest conversion tool for us so we just frame properly, research suggest is
enough safety" — and "I think you're playing it too safe on a lot of these"
(2026-05-04 review of Wave-5 Psoriasis pack). Attorney already cleared the
framing. The hedging-within-hedging pattern was killing the value of the
bold-claim conversion framework.

**How to apply:**
- Subtitles: lead with the bold outcome verb ("can reverse," "can drive
  remission," "research-described complete resolution") + single Path B hedge
- Outcomes tables: use the strongest research-described language; cite the
  paper/trial that backs it
- ONE critical-note section per pack, focused on the genuine hard
  contraindications (cancer, pregnancy, drug interactions) — not on
  "consult prescriber for every step"
- Don't pre-emptively undercut the framing with phrases like "not all
  cohorts" or "timeframes vary"
- Trust that the buyer can handle Path-B-hedged outcome language

**Citation discipline still strict:** paper / trial / research / regulatory only.
No creator names. That part of the editorial firewall does not loosen.
