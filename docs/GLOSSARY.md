# Glossary

Use these nouns in docs, scripts, and cockpit labels.

| Term | Meaning |
|---|---|
| Mission | One user objective with durable state and a next action. |
| Phase | One ordered step inside a mission. |
| Lane | A capability route to a tool or model family. |
| Receipt | Machine-readable evidence that a decision or action happened. |
| Artifact | A file produced or consumed by a phase. |

Avoid adding new nouns unless they name a distinct mechanism. Deprecated public
language: kernel, doctrine, wedge, DNA, operating system. Existing filenames may
remain until touched for behavior.
